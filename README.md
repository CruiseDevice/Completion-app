# Completion-app
A completion example using the OpenAI completion API.

<ins>Backend</ins>

- Create .env in root directory
- Put the following in .env

```
PORT=5000
BASE_URL=http://localhost:5000
OPENAI_API_KEY= <Your-OpenAI-key>
```

```
cd backend
npm install
node server.js
```

<ins>Frontend<ins>
```
cd frontend
npm install
npm run dev
```
